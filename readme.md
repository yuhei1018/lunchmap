lunchmap

https://laravel-lunchmap.herokuapp.com/shops

概要
自分のお気に入りのお店を投稿できる

※ポートフォリオ閲覧用としてトップ画面に”test user login”というボタンがあるので、
そのボタンを押すだけでもログインする事が可能です。(ご自由に編集や作成等行って頂いて大丈夫です。)

機能一覧

お店情報 / 作成・編集・削除
ログイン機能・ユーザー登録
